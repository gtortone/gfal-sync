# gfal-sync
GFAL sync utility

```
usage: gfal-sync.py [-h] [-c CONFFILE] [-r RECOVERFILE]

=== gfal-sync ===

optional arguments:
  -h, --help            show this help message and exit
  -c CONFFILE, --conf CONFFILE
                        JSON config file
  -r RECOVERFILE, --recover RECOVERFILE
                        JSON recover file
```
